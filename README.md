# Safer Calumpit: Crime Data Analysis Dashboard

## Project Overview
Safer Calumpit is a data analytics project focused on analyzing local blotter records to uncover crime patterns and trends. This project uses interactive dashboards to transform raw crime data into actionable insights that can support data-driven decision-making for public safety.

## Problem Statement
Crime incidents are often reviewed individually without identifying broader patterns. This project aims to analyze historical blotter records to detect trends based on location, time, and type of incident.

## Objectives
- Clean and preprocess blotter records
- Identify trends in crime occurrences
- Visualize high-risk areas and time patterns
- Provide insights for proactive public safety strategies

## Data Preparation
- Data cleaning and formatting in Microsoft Excel
- Handling missing values
- Standardizing crime categories
- Creating calculated fields for time-based analysis

## Dashboard Features
- High Risk Time Period
- Monthly and yearly trend analysis
- Location-based distribution of incidents/crimes
- Crime Frequency per Barangay
- Interactive filters and slicers for dynamic analysis

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Key Insights
- Certain crime types show recurring monthly patterns.
- Specific locations have consistently higher incident counts.
- Crime frequency varies depending on time period.
- Data visualization enables easier identification of high-risk trends.
